How to use Git:

Path for the project is https://github.com/Paraximia/IndianOstrich

Start with:

git clone url

Checkout the Development Branch, all changes are done here first and then tested before merged to master

git checkout Development 

*To switch between branches use, git checkout branchName


To add changes:

Step 1: Pull any new changes

git pull .

Step 2: Add any new files 

git add .

Step 3: Commit changes, DON'T FORGET TO ADD A MESSAGE

git commit -m "Your Message"

Step 4: Push Changes to git

git push
